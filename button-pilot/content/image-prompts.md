# AI photo prompts — pilot button heroes

The site ships with consistent **SVG button renders** (see `ButtonSwatch`), so it
looks complete today. When you want photoreal hero images, run the prompts below in
Midjourney / DALL·E / Firefly, then drop the results into
`public/images/products/<variantSku>.jpg` and wire them in (see README → "Adding
photoreal images").

**Keep all four consistent** — same background, lighting, framing, and crop — so the
catalog reads as one cohesive set. These are heritage metal workwear buttons, so lean
into honest patina and crisp stamped relief rather than glossy perfection.

## Shared template
> Studio product photograph of a single [SIZE-ligne] [TYPE] metal garment button,
> [FINISH] finish, centered on a seamless warm-grey (#ededea) background, soft diffused
> top-left lighting, subtle contact shadow, 1:1 macro, sharp focus, shallow depth of
> field, premium heritage-workwear catalog style, no text, no hands, no thread.

> finish, crisp raised lettering catching the light…

## Notes
- Generate 3–4 candidates per product; pick the one whose lighting/scale best matches the
  others.
- Brass/copper/nickel finishes should read as real metal — reflections and a little wear,
  not plastic.
- Disclose mockups vs. photos honestly; the footer already carries a "representative
  visuals, samples on request" line. B2B buyers expect sample cards.
