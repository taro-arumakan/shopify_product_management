# 月次レポート作成キット（フェーズ2：レポート生成用）

Drive にそろったシードCSVから月次レポートを作成する際の**参考資料**です。
シードファイルの準備（フェーズ1）は `reporter_bundle` 側で行います。ここはその次の工程。

## 中身
- **`monthly_report_file_specs.md`** … データ仕様書。
  各CSVファイルの構造・列の意味・どのレポート章に使うか、固定の算出ルール
  （売上＝純売上＋送料、AOV、ROAS＝CV目的のみ 等）をまとめたもの。
- **`monthly_report_prompt_JA.md`** … レポート生成プロンプトの**雛形（サンプル）**。
  そのまま使うものではなく、ご自身のプロンプトを作る・更新する際のたたき台。

## 使い方（フェーズ2）
1. 対象月の Drive フォルダ `Monthly Extraction / <YYYYMM> / <ブランド>` の
   CSV一式を Claude に渡す。
2. **プロンプトの中で `monthly_report_file_specs.md` に言及し、「まず読む」と指示**
   （ファイルを添付するか、Drive ルートにある同名ファイルを参照させる）。
   これで各数値の出典・算出ルールがそろい、生成品質が安定します。
3. `monthly_report_prompt_JA.md` を参考に、ご自身のプロンプトで生成・調整する。

## 補足
- `monthly_report_file_specs.md` は Drive の「Monthly Extraction」直下にも置いてあり、
  データと一緒に参照できます（このフォルダのものと同じ内容）。
- シードCSVの準備（IGストーリーズの分割、LINEのダウンロード等）は
  `reporter_bundle` 側のフェーズ1で完了している前提です。
